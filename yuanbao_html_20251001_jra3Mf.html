基于运算符、选择结构和循环结构的完整总结

Python知识点总结

1. 运算符

Python提供了丰富的运算符用于数据处理和逻辑判断：

算术运算符

用于数学计算：+（加）、-（减）、*（乘）、/（除）、//（整除，向下取整）、%（取余）、**（幂运算）

# 示例
print(9 % 2) # 输出 1
print(2**3) # 输出 8

赋值运算符

用于变量赋值：=、+=、-=、*=、/=、//=、%=

# 链式赋值和系列解包赋值
a = b = c = 10 # 链式赋值
a, b, c = 10, 20, 30 # 系列解包赋值

比较运算符

用于比较值：<、>、<=、>=、!=、==（值比较）、is、is not（身份比较）

布尔运算符

用于逻辑运算：and、or、not、in（成员检测）

位运算符

用于二进制操作：&（与）、|（或）、^（异或）、~（取反）、<<（左移）、>>（右移）

2. 选择结构（条件判断）

根据条件执行不同的代码块：

单分支结构

if 条件表达式:
    条件执行体 # 条件成立时执行

双分支结构

if 条件表达式:
    条件执行体1
else:
    条件执行体2 # 条件不成立时执行

多分支结构

if 条件表达式1:
    执行体1
elif 条件表达式2:
    执行体2
else:
    执行体3

嵌套if结构

if 外层条件:
    if 内层条件:
        内层执行体
    else:
        内层其他执行体
else:
    外层其他执行体

Pass语句

占位符，保证语法完整性，可用于if条件执行体、for-in循环体和函数体

3. 循环结构

用于重复执行代码块：

range()函数

生成整数序列：

range(10) # [0, 1, 2, ..., 9]
range(1, 10) # [1, 2, ..., 9]
range(1, 10, 2) # [1, 3, 5, 7, 9]（步长为2）

while循环

# 四步循环法示例
i = 1 # ①初始化变量
sum = 0
while i <= 100: # ②条件判断
    sum += i # ③循环体
    i += 1 # ④更新变量

for-in循环

遍历可迭代对象（字符串、列表、range等）：

for item in "Python":
    print(item) # 依次输出 P、y、t、h、o、n

循环控制语句

break：终止整个循环

continue：跳过当前轮次，进入下一次循环

else语句

当循环正常结束（未遇到break）时执行

嵌套循环

循环中嵌套另一个完整循环结构
